安装bootstrap #

  1. 先创建 .bowerrc 文件,并指定第三方前端库的安装路径
$ touch .bowerrc

{"directory":"./lib"}

$ bower init
$ bower install bootstrap --save

如果报git没有安装，需要把git的安装目录添加到环境变量中去,用分号隔开 C:\Program Files\Git\bin
